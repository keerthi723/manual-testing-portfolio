# Manual Testing Portfolio - TechHub

This repository contains manual testing documents for **TechHub**, a React-based online learning platform for IoT, Hardware, and Embedded Systems.

Live URL: https://tech-hub-sable-two.vercel.app

---

## About the Application

TechHub is an online education platform for learning IoT and hardware development. Key features include:

- Home page with hero section, image slider, and CTA buttons
- Explore dropdown menu with 4 course categories (IoT & Smart Devices, Hardware & Electronics, Robotics & AI, Embedded Systems)
- Search bar for finding courses and projects
- Signup page with Full Name, Email, Password fields
- Login page with Email, Password fields and social login (Google, GitHub, Twitter)
- Contact page with phone, email, location, and working hours
